# Retail Sales Dashboard | Power BI
# Project Overview

The Retail Sales Dashboard is an end-to-end data analytics project built using Microsoft Power BI.
This dashboard analyzes retail sales data to track sales performance, profit, quantity sold, top products, sales channels, and payment modes, helping management take data-driven business decisions.

The project covers the complete BI workflow including data cleaning, data modeling, DAX calculations, and interactive dashboard design.

# Key Insights

Online sales contribute the highest share of total revenue

Monthly sales show clear seasonal trends

A few top products generate maximum revenue

Cash and Online are the most used payment modes

Certain categories drive higher profit margins

# Dataset Overview

Retail transaction data (400K+ records)

File type: CSV

Key Columns

Order Date

Product ID

Category

Sales Type (Direct / Online / Wholesale)

Payment Mode

Quantity

Sales

Profit

# Data Cleaning & Transformation (ETL)

Performed using Power Query in Power BI:

Removed null and duplicate records

Corrected data types

Extracted Year, Month, and Day from Order Date

Standardized category and sales type values

# Data Modeling & DAX

Created a structured data model

Used DAX measures for KPIs

Key Measures

Top Selling Products
